MaGGie
======

![Maggie Simpson](http://upload.wikimedia.org/wikipedia/en/9/9d/Maggie_Simpson.png)

MaGGie (Molecule Geometry Generator) - program for building 3D geomtry from molecule structure.

Building
--------

    $ premake4 gmake
    $ make

Dependencies
------------

* `gsl`
* `sqlite3`
* `glib-2.0`
